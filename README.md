# Web Scraper Project

This project is a Python-based web scraper built using VS Code.

## Requirements
- Python 3.x
- VS Code
- Required libraries:
  - selenium
  - beautifulsoup4
  - requests

## Installation
Install dependencies using:
pip install -r requirements.txt


## How to Run
python scraper.py


## Output
The scraped data is saved in:


output.csv


## Notes
- Make sure ChromeDriver is installed if using Selenium
- Use delays to avoid getting blocked
- Scrape responsibly"# web-scraper"
